# wia-internal-mvp-test

Public **throwaway test site** for the Website Intelligence Agent (WIA) internal MVP.

- **Content is mock/demo only** — a fictional "Harbor Lantern Bakery". No real business, no client data, no credentials.
- **Purpose:** prove one safe edit-deployment loop end to end: one page → one edit → one QA → one deploy → one live verification → one rollback → one ledger update.
- Live page: https://barnicleproductions.github.io/wia-internal-mvp-test/

This repo is intentionally **separate from the private Second Brain vault**. Nothing sensitive is ever published here.
